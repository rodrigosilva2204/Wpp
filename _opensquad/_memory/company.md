# Perfil do Projeto

## Identificação
- **Nome:** Sistema de Promoções via WhatsApp (Wpp Afiliados)
- **Responsável:** Rodrigo
- **Uso:** Pessoal — envio de promoções para grupos de afiliados no WhatsApp

## Descrição
Sistema web que captura promoções do Mercado Livre e outras plataformas de e-commerce, copia automaticamente a mensagem e a imagem da promoção, substitui o link original pelo link de afiliado do usuário, e envia a mensagem formatada nos grupos de WhatsApp selecionados.

## Funcionalidades Principais
- Scraping/captura de promoções de plataformas (Mercado Livre, etc.)
- Cópia automática de mensagem + imagem da promoção
- Substituição do link pelo link de afiliado
- Login no WhatsApp via QR Code dentro do sistema
- Seleção de grupos para envio
- Padrão visual de mensagem fixo (baseado em exemplo de referência)

## Público-Alvo
Uso pessoal — grupos de afiliados no WhatsApp

## Stack Técnica
- Node.js (server.js já existente no projeto)
- Interface web

## Observações
- O usuário tem um link de exemplo com o padrão de mensagem a ser seguido (a ser compartilhado)
- As mensagens devem seguir o mesmo formato do exemplo de referência
